# 2K BeatBoard

**Integrated marketing operations cockpit — 2K Manager, Integrated Marketing — North America (interview demo)**

**🔗 Live:** [2k-beatboard.vercel.app](https://2k-beatboard.vercel.app)

A static, AI-pre-computed dashboard that gives a Marketing Manager a single operating picture across eight 2K franchises in live-service mode. Surfaces what's on fire today, what needs escalation up to Davide Detta (Sr Manager, Global GTM, Core), and what's queued for sign-off this week.

## Strategic brief

The work of a 2K Integrated Marketing Manager in NA is coordination under constraint — eight franchises, dozens of vendors, hundreds of Slack threads, transcripts, and budget tranches per quarter. The signals are real; the bandwidth to read them is the bottleneck.

**BeatBoard treats Claude as a build-time analyst, not a runtime chatbot.** Every transcript summary, every escalation draft, every "flagging for Davide" recommendation in this app was generated by Claude during the build and committed as JSON. The runtime is a pure React dashboard — zero AI API calls, zero latency, zero risk of a failed call mid-demo. The same pattern Alex used on Xsolla EventIQ, applied to a marketing-ops surface.

The Manager voice is execution-coordinator, not strategist. Alex **flags risks up**, **drafts comms for Davide's sign-off**, and **coordinates vendor pauses** — Davide makes the GO/NO-GO calls on budget tranche releases. That hierarchy is reflected in every escalation, every ticket, every "next milestone" line in the data.

## Modules

1. **Daily Standup** (`/`) — Claude-generated portfolio rollup, 6-KPI strip, by-title cards with phase-gate readouts, top risks with one-click escalation drafts, suggested escalations queue
2. **Global Calendar** (`/calendar`) — Custom Gantt across all 8 titles · beats or tasks view · title-filter pills · TODAY line at 2026-05-24
3. **Title Workspace** (`/titles`, `/titles/[slug]`) — Per-franchise drill-down across 6 tabs: overview, calendar, tickets kanban (P0/P1/P2 across open/in_progress/at_risk/completed), budget pacing, KPI compare, AI Inbox slice
4. **AI Inbox** (`/inbox`) — Five-tab unified queue: escalation Drafts (open modal with Edit + Send-to-Davide), Zoom Transcripts (expandable with extracted_tickets side panel), Slack, Gmail, Activity feed
5. **Connections** (`/connections`) — The six integrations Claude reads from (Slack, Gmail, Zoom, Steam Partner, Gamesight, Jira), with scope summaries and last-synced timestamps. Single source of truth for what the AI can see and do on the Manager's behalf.

## Architecture

- **Next.js 14** App Router (static export-ready)
- **React 18**, **Tailwind 3**, **Recharts** (sparklines are custom SVG), **lucide-react**
- All data in `/data/*.json` (13 files), pre-computed at build time
- Routing: `generateStaticParams` over the 8 franchise slugs
- Zero runtime AI dependencies — no API key required to run or demo
